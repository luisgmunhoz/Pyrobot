### Componentes

1. Raspberry Pi Pico

2. 3 Sensores de distância ultrassônico HC-SR04

3. 2 Motores DC

4. 2 Correias

5. Carcaça de tanque Robocore

6. Placa de ciruito impressa

7. Ponte H  L298N

8. 3 Pilhas de 3.7V e Suporte para pilhas

9. Jumpers

10. Paciência

11. Conversor de nível lógico

12. 2 cabo jacaré x jacaré
